## 概要
PHP/laravelの勉強として作成した、アパレルECサイトのポートフォリオです。 一般的なECサイトに実装される、基本的な機能を実装しました。

## バージョン
PHP 7.3.11
laravel 5.8.38

## 実装機能
### 商品関連 
・一覧表示機能<br>
・詳細表示機能<br>
・カテゴリー機能<br>
・ランキング機能<br>
・リコメンド機能<br>
・検索機能(JavaScript)<br>
・カート機能<br>
・お気に入り機能(JavaScript)<br>
・決済機能(Stripe)<br>

### ユーザー関連
・ログイン、ログアウト機能

### 管理者関連
・認証機能<br>
・一覧表示機能<br>
・編集機能<br>
・削除機能<br>

## 使用している技術
・AWSによる環境構築<br>
・データベースにはMySQLを使用<br>
